What is Zoho Catalyst ?

Zoho Catalyst is a highly comprehensive serverless development platform that can be used for power applications. It provides multiple serverless features that are considered important for building applications and other solutions.

In essence, Zoho Catalyst is a serverless web applications platform that enables specific vertical applications to get developed quickly, tested, deployed, and even supported without worrying about global standards. Essentially, it sits at the very core of the serverless strategy and gives unlimited flexibility for digitalizing multiple manufacturing processes.

Build Zoho Catalyst Project

The first step in developing a Catalyst application is to create a Catalyst project for it.

1. Log in to the Catalyst console and click Create a new Project.

2. Enter the project's name in the pop-up window.

3. Read and accept Zoho Catalyst's terms of service, then click Create.

4. Click Access Project to open it.

Install Catalyst CLI to develop, test, and deploy Catalyst applications from your local system. Refer to this help page for a detailed guide.

1. Execute this command in your terminal: 

npm install -g zcatalyst-cli

2. Execute this command to verify if the CLI is properly installed: 

catalyst --version

You will now be able to access the catalyst command from your terminal. Refer to the CLI help resources for detailed help with the CLI commands and actions.

1. Execute this command to log in to your Zoho account from the CLI: 

catalyst login

2. Type Y to allow Catalyst to collect CLI error reports and press Enter. 

$ catalyst login
Allow Catalyst to collect CLI error reporting information? (Y/n) : Y

3. Sign in using your credentials in the Zoho accounts login page.

4. Click Accept to provide data access permissions to the CLI.

5. Close the browser window and continue using the CLI in your terminal.
